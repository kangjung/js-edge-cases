자바스크립트의 예상치 못한 동작, 기묘한 형 변환(Type Coercion), 그리고 언어적 엣지 케이스들을 수집하고 분석하는 공간입니다. 개발자들이 자바스크립트의 '심연'을 재미있게 탐험할 수 있도록 제작되었습니다.

[👉 **실시간 웹사이트 구경하기**](https://kangjung.github.io/js-edge-cases/)

---

## 🧐 왜 이런 걸 만드나요?
`[] == ![]`가 `true`가 나오는 현상은 단순한 유머를 넘어 ECMAScript 명세에 근거한 동작입니다. 이 프로젝트는 다음을 목표로 합니다:
- 자바스크립트의 'WTF' 모먼트를 기록합니다.
- 왜 이런 결과가 나오는지 내부 로직을 설명합니다.
- 실무에서 발생할 수 있는 잠재적인 버그를 이해하도록 돕습니다.

## 🚀 주요 기능
- **인터랙티브 스포일러:** 정답을 보기 전 결과를 직접 예측해볼 수 있습니다.
- **다국어 지원:** 모든 케이스는 영어와 한국어로 기록됩니다.
- **데이터 기반 구조:** `cases.json` 수정만으로 누구나 쉽게 새로운 사례를 추가할 수 있습니다.

## 🛠 기여 방법
여러분이 알고 있는 '킹받는' 자바스크립트 코드가 있나요? 언제든 제보해 주세요!
1. 이 리포지토리를 **Fork** 합니다.
2. `cases.json` 파일에 새로운 케이스를 추가합니다.
3. **Pull Request**를 보냅니다.

자세한 규칙은 [CONTRIBUTING.md](./CONTRIBUTING.md)를 확인해 주세요.
